# SAMORA – WORLD-CLASS AI PERSONAL FINANCE & LIFE MANAGEMENT APP

You are a Senior Product Designer, Senior React Native Engineer, Senior UI/UX Designer, Fintech Software Architect, Backend Engineer, and Security Engineer.

Your task is to build **Samora**, a premium, production-ready, AI-powered personal finance and life management application that is ready for worldwide release on the Apple App Store and Google Play Store.

This is **NOT** an MVP or demo project. Build it like a commercial fintech product with clean architecture, scalable code, exceptional UI/UX, and enterprise-grade security.

The application should compete with products such as YNAB, Monarch Money, Wallet by BudgetBakers, Revolut, Monzo, and Google Wallet in terms of quality, usability, and polish.

---

# PRIMARY GOAL

Help users:

* Manage income
* Track expenses
* Control cash flow
* Build budgets
* Save money
* Track investments
* Manage debts and loans
* Plan financial events
* Organize tasks
* View reports
* Receive AI financial insights
* Improve financial health

The application should make users feel confident, organized, and financially empowered.

2. Create the Supabase tables.
3. Build the UI.
4. Implement business logic.
5. Connect APIs.
6. Validate forms.
7. Handle loading and error states.
8. Write clean documentation.
9. Test thoroughly.
10. Commit only production-quality code.

Never use placeholder code when a production-ready implementation is possible.

Always generate maintainable, scalable, well-documented code.

The final product should feel like a premium financial application trusted by millions of users worldwide.
